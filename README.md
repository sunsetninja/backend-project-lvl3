## Installation

```sh
  make setup
```

## Usage

To see a list of commands that PageLoader offers, you can run:

```sh
  page-loader --help
```

### Usage signature

```sh
  page-loader [options] <pageurl>
```

### Usage examples

```sh
  page-loader --output output-folder https://example.org
```
